# WHAT USE CANNOT DO

USE cannot give predictions. The metrics used in the sizing algorithm
are based on multiple inputs and averaged. The average of each metric
is used in the calculation, regardless of the deviation of the
sources. It is the responsibility of the user to consider the
customer’s environment and requirements when using this tool, and to
adjust both the input parameters and the provided results to meet the
customer’s expectations. Other scenarios USE 4.1 cannot handle are:

- Copilot sizing,
- Universal Shares,
- Performance for 5340-HA, Flex-5340-HA, Flex-5350-HA,
- Tape out, and
- MSDP-C for Flex Scale 5551, Flex Scale 5562
