# WHAT USE CAN DO

USE can take a set of workloads and constraints and create multiple
scenarios where those workloads are assigned to one or more NetBackup
Appliances and give aggregated estimates of how much of the hardware
resources are consumed by each workload over time. USE is highly
configurable to give the user the ability to customize the scenarios
the tool provides. USE provides a basic, easy to read output, with the
option of exposing additional screens with more detail.
